# Game Rating Server

Простий HTTP-сервер на Go для зберігання та отримання рейтингу гравців.

## Ендпоінти

### Додати рейтинг
`POST /rating`

`Content-Type: application/json`
```
{
    “name”: “Player”,
    “score”: 120
}
```

### Отримати рейтинг
```GET /rating?limit=10```

Параметр `limit` необов’язковий. За замовчуванням повертається 10 гравців.

## Безпека
Сервер перевіряє заголовок:
```
android-secret: <значення>
```
Значення має збігатися у змінній середовища `SECRET`.

## Запуск

1. Створити файл `.env` і додати:
```
SECRET=your_secret
```
2. Запустити сервер:
``` bash
go run main.go
```
